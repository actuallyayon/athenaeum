# Athenaeum

A seamless and modern web application designed to digitize the traditional library experience. Users can explore a vast collection of books, filter by categories, and borrow titles digitally.

## Live URL
[Athenaeum on Vercel](https://actuallyayon-athenaeum.vercel.app/)

## Key Features
- **Modern Design:** A highly polished, aesthetic interface with a premium feel.
- **Authentication:** Secure email/password and Google OAuth using BetterAuth and MongoDB.
- **Library Catalog:** Browse books by category, search by title or author.
- **Private Routes:** Secure routes for viewing book details and managing your user profile.
- **Profile Management:** Update personal information and profile picture.

## Technologies Used
- Next.js (App Router)
- Tailwind CSS & DaisyUI
- BetterAuth
- MongoDB
- SwiperJS
- React Toastify
